# Kanban-Board

## 소개
React로 접근성이 좋은 드래그 앤 드롭 리스트를 만들 수 있도록 도와주는 라이브러리를 공식 문서와 강의를 보며 스터디한 내용입니다.

## 스터디 노트
[react-beautiful-dnd](https://velog.io/@bomida/React-kanban-board)

## 개발 환경
  - 개발도구: VSCode, Github
  - 사용언어: React JS
